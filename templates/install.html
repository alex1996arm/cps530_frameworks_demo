Installation of our project

Since we were using github to track our development changes, the easiest way to host our application would be using Heroku. You can also host your application on heroku by uploading local files, which is what we're doing in this tutorial

first, you must enter your root directory and create a requirements.txt file
pip freeze > requirements
Then, you must create a Procfile in your root folder
echo "web: gunicorn myproject.wsgi" > Procfile
You must install django-heroku
pip install django-heroku
Then you must add import django to the top of your file
sed -i '1s/^ import django_heroku / settings.py'
Then you must this to the bottom of your settings.py file
# Activate Django-Heroku. django_heroku.settings(locals())

to create a new instance

You must have a heroku account. In a linux terminal.
sudo snap install heroku --classic
heroku login
heroku create
git push heroku master
heroku ps:scale web=1
heroku open

and you're done! wasn't that easy?
