How to create a great looking page with Materialize and Django

Materialize and django are a great combo, allowing you to create responsive, beautiful web applications with very little effort. Materialize is the front-end framework, whose skeleton consists of HTML, CSS and Javascript. Django is a backend framework that uses python. This tutorial will be divided into two parts, front end and back-end

Front-end

To create front-end skeleton code, you must first create an html file, so within the linux command line issue this command:

touch demo.html

Then you must open that demo.html file in the text editor of your choice and add the following elements

<!DOCTYPE html> <html> <head> <!--Import Google Icon Font--> <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"> <!-- Compiled and minified CSS --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css"> <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha256-3edrmyuQ0w65f8gfBsqowzjJe2iM6n0nKciPUp8y+7E=" crossorigin="anonymous"></script> <!-- Compiled and minified JavaScript --> <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script> <!--Let browser know website is optimized for mobile--> <meta name="viewport" content="width=device-width, initial-scale=1.0"/> </head> <body> <!-- Compiled and minified CSS --> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css"> <!-- Compiled and minified JavaScript --> <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script> </body> </html>

Back-End

Backend will be a little bit more involved, first of all, you must make sure python and django are installed. Once that is done, the first step is to type:

django-admin startproject mysite

Then, you're going to want to add url routes, views and templates. In the urls.py you must add

path('sitetuto/', views.sitetuto),

then you must create a folder called templates in the root folder of the project and add you front-end code to a template you will call sitetuto.html, finally you must create a views.py file in the root directory and add the following to views.py

from django.http import HttpResponse
from django.shortcuts import render

on the top and lower down:

def sitetuto(request):
return HttpResponse(render(request, 'sitetuto.html'))
